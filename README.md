# Immutable-wallet

Wallet library using immutable data structures and typescript.

  * typescript
  * immutablejs
  * mocha
  * chai and chai-immutable
  * tsd

## Usage

```
1. npm install
2. npm start
3. npm test
```

## requires

```
npm install -g typescript
npm install -g tsd
```

## IDE

You can use Atom with `atom-typescript`

```
apm install atom-typescript
```
